# Beirut Unlocked  

A web application that helps users discover and explore various locations in Beirut, Lebanon.  

## Features  
- Interactive map interface  
- Categorized locations:  
  - Hotels  
  - Restaurants  
  - Coffee Shops  
  - Museums  
- Recently viewed locations  
- Ride services  
- Help desk support  

## Technologies Used  
- HTML5  
- CSS3  
- JavaScript  
- SweetAlert2 for notifications
